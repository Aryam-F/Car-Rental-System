# Car-Rental-System
This project is designed to provide an efficient and user-friendly application for customers to view and rent cars. The program, implemented in assembly language using EMU8086.

Features a menu-driven interface that simplifies the car rental process. Users can easily access information about available vehicles, make bookings, and calculate rental costs, including discounts for members.
